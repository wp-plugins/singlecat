== Description ==

A simple plugin that displays a specified number of posts from a category using shortcodes.  The category list can be made public or private.  The results are not paged.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[singlecat cat=(CAT#) level=(public or private) posts=(#ofPosts)]` in your posts, pages, or text widgets.
1. Example: `[singlecat cat=23 level=private posts=10]`
